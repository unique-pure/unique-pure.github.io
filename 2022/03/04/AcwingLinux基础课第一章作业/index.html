创建作业 & 测试作业的正确性

  • homework 1 create可以重新创建所有lesson_1的作业。

    • homework 1 create id可以单独创建lesson_1的第id个作业。

      如homework 1 create 0可以单独创建lesson_1的第0个作业。

  • homework 1 show 可以显示第lesson_1的作业要求。

  • homework 1 test可以评测lesson_1的所有作业。

第一章作业、常用文件管理命令

==创建好作业后，先进入文件夹/home/acs/homework/lesson_1==。然后开始我们的作业。

此指令为cd /home/acs/homework/lesson_1（绝对路径）或cd homework/lesson_1（相对路径）。

  1. 进入homework_0文件夹，分别创建文件夹dir_a,dir_b,dir_c。

    指令步骤：

    • cd homework_0。
    • mkdir dir_a，同理创建其余两个。==当然，linux也支持同时创建，即我们直接输入mkdir dir_a dir_b dir_c==即可完成。

  2. 进入homework_1文件夹，将a.txt,b.txt,c.txt分别复制成：a.txt.bak,b.txt.bak,c.txt.bak。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_1或者cd ../homework_1。
    • cp a.txt a.txt.bak，同理复制其余两个。==linux==支持同时复制多个文件，使用花括号即可。

    ==如复制错了，需要删除，我们可以使用rm命令，使用通配符可将匹配的文件都删除==。

  3. 进入homework_2文件夹，将a.txt,b.txt,c.txt分别重命名

  4. 成：a_new.txt,b_new.txt,c_new.txt。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_2或者``cd ../homework_2`
    • mv a.txt a_new.txt，同理重命名其余两个。

  5. 进入homework_3文件夹，将dir_a文件夹下的a.txt,b.txt,c.txt分别移动到文件夹dir_b下。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_3或者cd ../homework_3。
    • mv dir_a/a.txt dir_b/，同理移动其余两个。==Linux提供了一种更快捷的方式，即mv dir_a/{a.txt,b.txt,c.txt} dir_b/==。

  6. 进入homework_4文件夹，将普通文件a.txt,b.txt,c.txt都删除。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_4或者cd ../../homework_4。
    • rm a.txt -rf，同理处理其余两个。我们也可以rm {a.txt,b.txt,c.txt} -rf。

  7. 进入homework_5文件夹，将文件夹dir_a,dir_b,dir_c都删除。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_5或者cd ../homework_5。
    • rm {dir_a,dir_b,dir_c} -rf 。

  8. 进入homework_6文件夹，查看task.txt的内容，并按指示操作。该指示为：将task.txt重命名为done.txt，创建目录dir_a，将done.txt移动到目录dir_a下。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_6或者cd ../homework_6。

    • cat task.txt。查看文件内容

    • mv task.txt done.txt。重命名。

    • mkdir dir_a。创建目录dir_a。

    • mv done.txt dir_a/。移动到dir_a目录下。

  9. 进入homework_7文件夹，创建文件夹dir_0,dir_1,dir_2。

    将a.txt,b.txt,c.txt复制到dir_0下，重命名为a0.txt,b0.txt,c0.txt。

    将a.txt,b.txt,c.txt复制到dir_1下，重命名为a1.txt,b1.txt,c1.txt。

    将a.txt,b.txt,c.txt复制到dir_2下，重命名为a2.txt,b2.txt,c2.txt。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_7或者cd ../homework_7。
    • mkdir dir_0 dir_1 dir_2。创建文件夹。
    • cp a.txt dir_0/a0.txt。复制并重命名，其他任务同理。

  10. 进入homework_8文件夹，分别在dir_a,dir_b,dir_c文件夹下查看task.txt的内容，并分别按照指示操作。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_8或者cd ../homework_8。

    • cd dir_a。进入dir_a文件夹。

      • cat task.txt。查看文件内容。其指示为将a.txt删除。
      • rm a.txt。删除a.txt。

    • cd ../dir_b。进入dir_b文件夹。

      • cat task.txt。查看文件内容。其指示为将b.txt重命名为b_new.txt。
      • mv b.txt b_new.txt。重命名b.txt。

    • cd ../dir_c。进入dir_c文件夹。

      • cat task.txt。查看文件内容。其指示为将c.txt复制成为c.txt.bak。
      • cp c.txt c.txt.bak。复制。

  11. 进入homework_9文件夹，将其中所有txt类型的文件都删除。

    指令步骤：

    • cd /home/acs/homework/lesson_1/homework_9或者cd ../../homework_9。
    • rm *.txt -rf。删除。Linux的通配符规则如下：*代表任意字符(0到多个)；?代表一个字符；[]中间为字符组合，仅匹配其中任意一个字符。
